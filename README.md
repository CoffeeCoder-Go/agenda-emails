# Agenda Emails
É um projeto de uma agenda de emails que contém diversos recursos como:

- Gerenciamento de emails(CRUD)
    - Pegar pelo id
    - Listar todos
    - Criar email com apelido
    - Deletar email
    - Mudar apelido
- Ordenação
- Mandar emails
- Pesquisa
- Classificação por cor(Você pode criar a classificação) 

## Como rodar
Rode:
    make run

Pronto!
